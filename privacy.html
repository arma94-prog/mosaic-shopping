2.4 행동 분석 데이터 (Mixpanel)

본 확장은 익명화된 행동 로그를 Mixpanel(미국 소재)을 통해 분석합니다. 이 분석은 Google 계정 연결 여부와 무관하게 본 확장의 모든 사용자에게 기본 적용됩니다.

항목이용 목적보유 기간
검색/북마크/클릭 이벤트 로그기능 개선, 사용 패턴 분석Mixpanel 정책에 따름
익명 ID(distinct_id)동일 사용자 세션 연결Mixpanel 정책에 따름
행동 분석 데이터에는 검색어 원문이 포함됩니다 (예: "에어팟"). 다만 사용자의 이메일/이름/주소 등 직접 식별자는 포함하지 않습니다. Mixpanel은 분석 목적으로만 사용되며, 광고나 제3자 마케팅 목적으로 활용되지 않습니다.

2.5 쿠팡 파트너스 어필리에이트 변환

본 확장은 쿠팡 상품 URL을 어필리에이트 추적이 가능한 단축 URL로 변환합니다. 이 과정에서 사용자의 식별 정보는 쿠팡에 전송되지 않으며, 상품 URL만 변환 목적으로 일시 처리됩니다.

4. 사용자 권리와 행사 방법

4.1 동의 철회 / 데이터 삭제

옵션 페이지의 "연결 해제" 버튼을 클릭하면, 다음이 순차적으로 즉시 실행됩니다:

  1. 클라우드(Supabase)에 저장된 본인의 모든 미러 데이터 삭제 — 북마크 그룹, 북마크 항목, 핀고정 키워드, 검색 이력, 사용자 환경설정.
  2. Supabase 측 인증 세션 종료 — 발급된 토큰 무효화.
  3. PC의 인증 토큰 삭제 — 더 이상 클라우드 호출 불가.
v0.3(v1.24.2)에서 1단계 삭제 대상에 검색 이력과 사용자 환경설정이 추가되었습니다.

예외 처리: 1단계가 네트워크 오류 등으로 실패한 경우에도 2-3단계는 진행됩니다(토큰 끊기 자체는 무조건 성공). 1단계 실패 시 클라우드 데이터가 남을 수 있으므로, 인터넷 환경 복구 후 재연결 → 재해제 또는 arma94@gmail.com으로 메일 주시면 3영업일 내 수동 처리합니다.

4.2 데이터 열람 / 정정

본 확장의 사이드 패널 및 옵션 페이지에서 직접 본인 북마크/키워드/환경설정을 열람·정정·삭제할 수 있습니다. 추가 정보 열람을 원하실 경우 위 이메일로 연락 부탁드립니다.

4.3 행동 분석 거부

Mixpanel 행동 분석은 현재 별도 거부(opt-out) UI가 제공되지 않습니다. 거부를 원하실 경우 본 확장 사용을 중단해주시기 바랍니다. Phase 2에서 거부 토글 추가 예정.
